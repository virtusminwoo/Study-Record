실행법 

Node.js (v6.10.2)
 
 
터미널에
 
git clone https://github.com/virtusminwoo/Study-Record.git 
 
입력.

 
Study-Record폴더가 생김. 
 
 
Study-Record폴더로 이동
 
터미널에

npm install
 
입력시 module들이 설치.
 
 
module설치 후
 

터미널에
 
yarn start
 
입력시 자동으로 실행.
